第一步：修改程式碼
在您的電腦上打開 src/App.js，進行您想要的修改或是功能新增，然後儲存檔案。

(建議先在電腦上執行 npm start 確認修改後的功能是正常的)

第二步：上傳更新 (Git Push)
打開終端機 (Terminal)，確認路徑在您的專案資料夾內 (mobile-mo)，然後依序輸入以下三個指令：

1.加入修改檔案

git add .

2.提交修改並寫下備註 (引號內的文字可以隨意寫，例如 "新增刪除功能" 或 "修正錯字")

git commit -m "update app"

3.推送到雲端

git push

第三步：等待自動更新
當您執行完 git push 並看到上傳成功的訊息後：

GitHub 會收到最新的程式碼。

Vercel 會自動偵測到有新東西，並馬上開始重新部署。

大約等待 1~2 分鐘，您手機原本那個網址的內容就會自動變成最新的版本了！(如果沒變，重新整理一下手機網頁即可)

總結
以後每次改完程式，就是這三招： git add . -> git commit -m "..." -> git push 收工！
